# Installation instructions:
Debian and debian based systems:
```
git clone https://github.com/Selector0073/ezgit && cd ezgit && sudo bash ./installation
```

# To get help type:
```
ezgit --help
```
or
```
ezgit <--command> <path>

--pull:
ezgit --pull username/project

--clone:
ezgit --clone username/project

--cap:
Commit and pull

--login

--help
```
